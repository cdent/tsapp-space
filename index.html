tsapp

tsapp is a command line tool for creating browser-based applications that use TiddlySpace as the host for content and data. It was used to create this page. To see the source of the tsapp directory that was created visit the GitHub repo.

If you have a working Python installation you can install or upgrade with pip install -U tsapp. See the tsapp repo for more details.

Commands

init

Create the directory and stub files for a new app.

tsapp init <directory name>
auth

Authenticate the provided user against the target_server.

tsapp auth <username>
push

Push the assets to the target server, into the target_bag. If the bag doesn't end in _public or _private then _public will be aded.

tsapp push <target_bag>
serve

Run a WSGI server that proxies requests looking both locally and on the target server for resources.

tsap serve
delete

Delete a single tiddler from a bag on the target server.

tsap delete <bag_name> <tiddler_title>

Configuration

tsapp looks for a .tsapp configuration file in the current user's $HOME and in the current directory. $HOME/.tsapp overrides defaults and ./.tsapp overrides $HOME/.tsapp`.

target_server

The default server where content will be pushed and proxied.

Default is http://tiddlyspace.com.

local_host

The hostname of IP to which to bind the local proxy.

Default is 0.0.0.0.

port

The port to which to bind the local proxy.

Default is 8080.

auth_token

Authentication secret for accessing private resources on the target server. Fill by the auth.
